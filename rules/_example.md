# Example Rule / 示例规则

This is an example rule file. Files starting with `_` are ignored by the rules loader.
To create your own rules, create a new `.md` or `.txt` file in this `rules/` directory.

---

这是一个示例规则文件。以 `_` 开头的文件不会被规则加载器读取。
要创建自己的规则，请在 `rules/` 目录下新建 `.md` 或 `.txt` 文件。

## 示例

```
# my_style.md
请使用中文回复。
节点命名使用下划线风格，如 terrain_generator。
VEX 代码中添加中文注释。
```
